## Notes:
- In Python, ```2 + 2``` is considered an *expression*.
  - Expressions have *values* (2) and *operators* (+)
- *Data type* - category for values; every value belongs to one data type
- String concatenation:
```
>>> 'A' + 'B'
'AB'
```
